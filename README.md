# 🎵 Music App with Android Studio

A simple and modern Android Music Player App built using Android Studio.  
This project allows users to play music smoothly with an easy-to-use interface.

## 📱 Features

- Play audio/music files
- Pause and resume songs
- Next / Previous track controls
- User-friendly UI design
- Smooth music playback experience

## 🛠️ Built With

- Java / Kotlin (depending on your codebase)
- Android Studio
- XML Layouts
- MediaPlayer API
